Design Notes
Shared Primary Color
#007AFF is the shared primary with the PC desktop app, ensuring visual brand consistency across platforms.
iOS Human Interface Guidelines
All semantic colors (#30D158, #FF9F0A, #FF453A) match iOS system colors for visual consistency and future Dynamic Color support.
Accessibility
Primary text on backgrounds meets WCAG AA (4.5:1+). Semantic colors are never the sole indicator of state — always paired with icons or text.
